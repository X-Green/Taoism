# HUMA3810 第 2 课整理：导论一，什么是“古代中国哲学”

## 本课主题

本课开始进入道家思想导论。老师先把“道家思想”放在“古代中国哲学”这个大框架下说明：道家或老庄思想，首先是一种哲学；其次属于中国哲学；再进一步，它是先秦、春秋战国以来的古代中国哲学。整堂课围绕这三个层次展开：

1. 什么是哲学；
2. 中国哲学相对于西方哲学的特点；
3. 为什么今天还要读古代中国哲学。

课堂没有正式细读《道德经》或《庄子》篇章，但多次预告道家会涉及“无用之用”、生命自由、逍遥自在等主题。

## 课堂结构

- 【课程事务/要求】承接第一课，进入导论。
- 【文本解释】说明“古代中国哲学”这一课程关键词的三层含义。
- 【思想解释】解释哲学、中国哲学和古代哲学的意义。
- 【历史回顾】介绍轴心时代、古今连续和传统思想的普遍价值。
- 【讨论问答】整理课堂中学生对哲学定义的回答。

## 详细整理

### 一、课程事务与本课定位

上一堂课主要处理 course outline、课程内容和要求，相关资料已放在 Canvas。本课若没有课程要求方面的问题，就正式开始导论。

本课所讲的“道家与中国传统”可先理解为一门关于道家思想、尤其是老子和庄子思想的课。老师指出，道家思想从最表面的层次看，是一种哲学；它属于中国哲学；由于老子、庄子主要属于先秦春秋战国时期，因此又属于古代中国哲学。

“古代中国哲学”这个词包含三层意思：哲学、中国哲学、古代中国哲学。本课依次解释这三层。

### 二、什么是哲学：从“思想”到“爱智”

老师先说明，很多同学中学阶段未必接触过哲学，因此需要先交代“哲学”是什么。课堂中有同学提出几个理解方向：哲学是探求人生真相与智慧；哲学是热爱智慧；哲学需要逻辑；哲学思考自我与世界的关系。老师肯定这些回答，但进一步区分“思想”和“哲学”。

“思想”范围很广，任何 thinking 都可以叫思想；“哲学”则更集中，关心更深入、更根本的问题。哲学没有一个所有学者都一致接受的标准定义，但仍有基本含义，不是可以随意乱说。

老师给出第一个简明解释：哲学就是“爱智”或“爱智慧”。中文“哲学”来自 philosophy，而 philosophy 的源头是古希腊文 philosophia，意思是爱智慧。这里的智慧不是一般知识，而是能够理解宇宙人生真理，并把这种理解落实到生活实践中的智慧。哲学不一定像会计、电脑、工程那样“实用”，但它不是无用；它关乎如何以智慧面对人生的困难、烦恼和痛苦。老师借用道家将来会讲到的说法，指出哲学可理解为一种“无用之用”。

### 三、哲学的较完整定义：以理性的态度反思宇宙人生的根本问题

老师给出第二个较完整的解释：哲学是“以理性的态度反思宇宙人生的根本问题”。

“宇宙人生”包括个人生命、家庭、社会、国家、民族、世界乃至整个宇宙。人在其中会面对许多问题，如学业、工作、爱情、经济、政治、社会等。哲学并不是处理所有问题，而是处理其中较根本的问题。

“根本问题”不是绝对概念，而是相对概念。经济问题、就业问题在现实处境中也可能非常根本；但哲学通常从更 general 的层面追问，例如：

- 生死问题：我从何而来，死后是否存在；
- 生命意义问题：我为什么活着，我生命的价值在哪里；
- 本体论或形而上学问题：宇宙背后的根源、本质或本体是什么；
- 宇宙论问题：宇宙如何形成和发展；
- 知识论问题：什么是正确知识，如何获得正确知识；
- 伦理学或道德哲学问题：什么是善恶、对错，应当做什么，不应当做什么。

老师进一步解释“理性的态度”。理性相对于主观、感性或情绪化的态度。感情用事可能妨碍人看清事物真相，但老师也补充，感性本身不一定不好；诗人、文学家、艺术家也能以敏锐感受把握人生和世界的深刻问题。哲学不是完全没有感情，而是其主要方式偏重理性。

理性的态度包括：提出主张时要给理由、证据或论证；思考不能违反逻辑；要有系统性；也要有自我反省。老师用逻辑三律说明理性思维的基本规矩：同一律、矛盾律、排中律。又用“戴红色眼镜看世界”的例子说明，自我反省能帮助我们检讨主体自身的预设，否则主体的预设会扭曲对对象的理解。

“反思”不是顺着已有结论继续往前推，而是回头追问问题的根源。老师举“我为什么不快乐”的例子：从考试不及格，到不能毕业、找不到工作、没有钱、挨饿乃至死亡恐惧，一层层追问，最后找到问题的根源。哲学式反思就是这种刨根问底。

### 四、中国哲学的特点：生命的学问

第二个层次是“中国哲学”。老师指出，如果每个人都可以有自己的宇宙观、人生观，那么不同社会、国家、民族也可以形成各自的哲学传统。课堂举到的哲学传统包括中国哲学、西方哲学、印度哲学、阿拉伯哲学、日本哲学、非洲哲学等。其中中国哲学、西方哲学和印度哲学常被视为较重要、较成体系的传统。

相对于西方哲学，中国哲学的特点可概括为：西方哲学较重视“自然”，中国哲学较重视“生命”。

西方哲学早期常从对自然宇宙的惊叹出发，追问宇宙的秩序、根源和真相。中国哲学则更关心人的精神生命如何安顿。这里的“生命”不是肉体生命，不是医学意义上的生命，而是精神生命。

老师用儒、释、道三家说明“中国哲学是生命的学问”：

- 儒家重视仁义道德，认为成就道德人格才有真正幸福；
- 佛教关心生命痛苦，追求从痛苦中解脱；
- 道家关心人生束缚与不自由，追求从束缚中解放，获得精神自由，尤其在庄子那里表现为逍遥自在。

老师特别提醒，这种概括只是重点差异，不是绝对排除。西方哲学也谈生命，中国哲学也谈自然；差别在于主导关怀和 priority 不同。

### 五、为什么读古代中国哲学

第三个层次是“古代中国哲学”。老师回应一个可能的问题：既然现代中国哲学似乎更贴近现实，为什么还要读老子、庄子这些两千多年前的古代思想？

老师给出三点理由。

第一，以古鉴今。历史像一面镜子，保存了人类过去的经验、成功与失败。好的经验可以借鉴，错误的经验可以警惕。古代思想虽然产生于过去，但仍能为今天提供参考。

第二，古不离今。古代与现代、传统与现在不是截然二分，而是一个连续体。老师用心理治疗作比喻：要理解一个人的当前问题，往往需要回溯童年经验和生命历程；要理解今天的文化和思想问题，也需要回到过去。老师还提到“唯有了解我们的过去，我们才能够立足于现在；唯有能够立足于现在，我们才能够展望我们的将来”。

在这一点下，老师引用雅斯培（Jaspers）的“轴心文明”或“轴心时代”说法：在人类历史某一时期，不同文明中几乎同时出现了一批重要思想家，他们奠定了后世文明的基础。课堂提到希腊哲学、苏格拉底，印度传统和释迦牟尼，中国的孔子和老子。老师也提到一种说法：西方哲学可看作对柏拉图哲学问题的不断注解；现代中国哲学也仍在不断解释和发展古代中国哲学。

第三，古代哲学有普遍价值。哲学观念当然产生于特定时代和文化背景，但一旦形成，也可能具有普遍意义和永恒价值。老师用孔子的“仁”和西方宗教哲学中的“上帝”作例子：这些概念有其特殊文化脉络，但也包含超出特定文化的普遍意义。道家思想同样如此，虽然产生于古代，但其中关于生命、自由、安顿等问题的思考，今天仍有意义。

老师最后补充一个现实原因：课程开设的是古代中国哲学或道家思想课程，所以同学实际修读的就是这一部分。

## 关键概念

| 概念 | 课堂解释 |
| --- | --- |
| 哲学 | 没有唯一标准定义，可简要理解为“爱智慧”，也可理解为“以理性的态度反思宇宙人生的根本问题”。 |
| 爱智 / 爱智慧 | philosophy / philosophia 的本义。智慧指理解宇宙人生真理，并能落实到生活实践。 |
| 无用之用 | 课堂借用道家说法说明哲学“不一定实用，但不是没有用”；后续课程会再讲。 |
| 宇宙人生 | 从个人生命到家庭、社会、国家、民族、世界、宇宙的一切。 |
| 根本问题 | 相对而言较 foundational 的问题，如生死、生命意义、本体、知识、善恶等。 |
| 理性的态度 | 有理由和证据，不违反逻辑，具有系统性，并能自我反省。 |
| 反思 | 回头追问问题源头，而非只顺着现有前提继续推论。 |
| 中国哲学 | 相对于西方哲学，课堂概括其重点为重视精神生命的安顿。 |
| 生命的学问 | 中国哲学主流关心如何安顿生命；儒家以道德安顿，佛教以解脱痛苦安顿，道家以自由逍遥安顿。 |
| 以古鉴今 | 从历史经验中照见当下，吸取经验与教训。 |
| 古不离今 | 传统与现代不是断裂的，而是连续体。 |
| 轴心文明 / 轴心时代 | 雅斯培提出的说法，指人类文明某一关键时期出现奠基性思想家和思想传统。 |
| 普遍价值 | 哲学观念虽有特定历史文化背景，但也可能具有超越时代的普遍意义。 |
