# 第 04 课

## 本课主题

本课从上一课对「道家在中国哲学与文化中的地位」的回顾，转入新主题：道家思想产生的背景。课堂重点不是直接细读《逍遥游》或《养生主》，而是说明道家为何会在春秋战国诸子百家并起的时代出现，并讨论几种解释路径：传统的「诸子出于王官」、冯友兰的修正、胡适的反对、牟宗三对「周文疲弊」的补充，以及南方楚地文化对道家气质的影响。

## 课堂结构

1. 【课程事务/要求】说明考试范围、导论部分的重要性、下次课安排。
2. 【文本解释】用杜甫、李白和《三国演义》开篇词补充道家式人生态度。
3. 【历史回顾】介绍「诸子出于王官」的传统说法及《汉书·艺文志》的来源。
4. 【思想解释】说明道家为何被说成「出于史官」。
5. 【讨论问答】回应学生关于能否从孔子、老子生平判断思想来源的问题。
6. 【思想解释】比较章太炎、冯友兰、胡适、牟宗三对王官说的不同态度。
7. 【历史回顾】以「周文疲弊」「礼崩乐坏」解释诸子百家兴起的共同背景。
8. 【思想解释】比较儒、道、墨、法对礼乐制度危机的不同回应。
9. 【历史/文化解释】说明南方楚地文化如何影响道家思想的表达风格。

## 详细整理

### 【课程事务/要求】考试范围与下次安排

老师说明，本课关于「道家思想产生的背景」属于导论部分。课程从第一堂到最后一堂都在考试范围内，但导论部分相对没有后面进入文本与思想细节时那么重要。

课末说明：下次会先把本课主题稍作总结，再继续讲后面的内容；因为退选期结束后，老师会说明报告如何准备。下个礼拜放假，停课一周，下下个礼拜再见。课堂最后有新年问候。

### 【文本解释】道家式人生态度的补充例子

课堂开头回顾上一课：道家在中国哲学和文化中的地位非常重要。先秦「九流十家」中，儒家和道家是影响最大的两家；佛教传入之后，儒、释、道成为中国思想文化的三大主流之一。道家对政治军事、宗教思想、文学艺术和中国人的人生态度都有影响。

老师回到上一课提过的杜甫与李白对比。杜甫体现儒家式忧国忧民，如关心朝廷、国家和百姓；李白面对人生不称意时则倾向离开、归隐和超脱，体现出较明显的道家情怀。

老师又补充《三国演义》开篇词：「滚滚长江东逝水，浪花淘尽英雄。是非成败转头空。青山依旧在，几度夕阳红。」这首词把英雄、是非、成败放在长江、青山、夕阳等长久景象中观看，显示历史功名终会过去。「古今多少事，都付笑谈中」进一步说明，许多曾经重大的事情最后不过成为后人的谈资。老师借此说明，道家式情怀会把人生看得较为旷达，不必过度执着于一时的是非、成败和功名。

这一段属于文本解释，同时也是从文学例子引出思想解释：道家的人生态度不是道家思想的全部，但它确实是道家影响中国文化的一面。

### 【历史回顾】传统说法：「诸子出于王官」

本课正式的新问题是：道家思想怎样产生？由于道家与其他诸子百家差不多同时出现在春秋战国，所以讨论道家背景时，也必须先说明诸子百家的共同兴起背景。

传统说法是「诸子出于王官」。老师说明，这个说法最早由汉代刘歆在《七略》中提出；《七略》今天已不易见到，但班固《汉书·艺文志》继承并保存了这个讲法。刘歆的父亲刘向也是汉代重要学者；老师顺带提到钱穆曾写《向歆父子年谱》，说明刘向、刘歆父子在学术史中的重要性。

「王官」指周代贵族官僚体系中掌握、保存和传承专门知识的官员。老师先说明西周、东周及春秋战国以前的社会背景：社会大致分为贵族与庶民两个阶层，政治、经济、文化知识等资源主要由贵族掌握。不同王官负责不同专业知识，因此传统说法认为，各家思想就是从这些专门知识中发展出来的。

### 【历史回顾】《汉书·艺文志》中的对应关系

课堂较完整地列出传统王官说的对应关系：

| 学派 | 传统所说来源 | 课堂解释 |
| --- | --- | --- |
| 儒家 | 司徒之官 | 掌管邦教、教育，发展出教人做人、成就道德人格的思想。 |
| 道家 | 史官 | 掌管历史、典籍，能够观察社会与文明发展的规律，由此思考「道」。 |
| 墨家 | 清庙之守 | 与宗庙、祭祀相关的职官；课堂只简述，不展开细节。 |
| 法家 | 理官 | 熟悉法律、刑法条文的官员，发展出法家思想。 |
| 名家 | 礼官 | 熟悉礼制、名分、仪式差别，可能由分辨礼制差别发展出名辩、逻辑方面的思想。 |
| 阴阳家 | 羲和之官 | 掌管天文知识，由观察宇宙而发展出阴阳五行解释。 |
| 纵横家 | 行人之官 | 出使、外交的官员，发展出外交游说之术。 |
| 杂家 | 议官/谏官 | 劝谏君主须综合多方面知识，故发展出杂家。 |
| 农家 | 农稷之官 | 掌管农业知识，发展出重农、以农立国等思想。 |
| 小说家 | 稗官 | 小官，采集民间意见、见闻与智慧，提供统治者参考。这里的「小说家」不是现代写小说的人。 |

老师强调，对本课主题最重要的是道家与史官的关系。史官掌管历史，能够从人类社会、文明兴衰和制度变化中理解背后的规律、法则或原则，这就是道家所谓「道」的一个来源。因此在传统说法下，道家思想产生的第一个背景就是「来自史官」。

### 【讨论问答】能否从孔子、老子生平判断来源？

课堂中有学生提问：既然孔子、老子等著名人物有生平记载，能否直接从他们的生平判断他们到底出于哪里？

老师回应说，王官说讨论的是诸子百家思想的来源，不一定能直接从每个人的生平材料中找到完整证据。有时只能依靠旁证。后面谈冯友兰时可以看到：孔子有贵族背景，后来成为民间教师；老子本来与史官身份有关，后来又成为隐者。这些材料可以帮助理解王官知识如何流落民间并发展成思想，但不能简单把「某个人的传记」等同于「整个学派的起源证明」。

### 【思想解释】近现代学者对王官说的态度

老师区分了几类态度。

章太炎大体赞成「诸子出于王官」的传统说法，并且提出新的证据支持其合理性。

冯友兰对传统说法作修正。他并不完全反对王官说，而是补充一个社会背景：从西周到东周、春秋战国，贵族与庶民之间原本清楚的阶层界线逐渐模糊；许多原本属于贵族、官府系统的人和知识流落民间，形成民间的读书人、教师、士人或知识分子。诸子百家不是仍在王官制度内部直接产生，而是由王官知识进入民间后，在士人阶层中发展出来。

冯友兰也给出一套「民间知识分子类型」的对应关系。课堂只列他认为较重要的六家：儒家来自文士，道家来自隐士，墨家来自武士，法家来自法术之士，名家来自辩士，阴阳家来自方士。老师特别解释隐士：古代隐士不是普通「躲起来的人」，而多是有学问、有智慧、看出时代无道后选择隐居的人。老子、庄子都可放在这一脉络中理解。

胡适则反对传统王官说，写有《诸子不出于王官论》。老师概括胡适的两个重点：第一，王官所掌握的专业知识与各家思想之间没有必然关系，不能从某种官职知识逻辑地推出某种学派思想。例如墨家由清庙之守发展出来、名家由礼官发展出来，在逻辑上都不够明显。第二，胡适正面提出，诸子思想来自「救世之弊」，即为了回应当时社会问题和民生疾苦而出现。

牟宗三对胡适作进一步修正。老师说明，胡适的「救世之弊」有道理，但太笼统：许多时代、国家和民族都有社会问题，为什么偏偏在中国春秋战国出现诸子百家？牟宗三认为要指出具体问题，也就是「周文疲弊」。同时，牟宗三还区分「逻辑意义的出」和「历史意义的出」：胡适反驳的是不能从王官职能逻辑推出诸子思想；但如果把「出于」理解为历史来源，即某类思想事实上由某类具有专业知识背景的人发展出来，那么传统说法未必错误。

### 【历史回顾】「周文疲弊」与礼崩乐坏

「周文疲弊」是牟宗三用来补充胡适「救世之弊」的概念。「周文」指周代文化、典章制度，尤其是支撑周代文明的礼乐制度；「疲弊」就是出现流弊、衰败和失效。

老师回顾周公制礼作乐，奠定周代文化基础。到了东周、春秋战国，礼乐制度出现崩塌，即常说的「礼崩乐坏」。这不只是仪式形式出现问题，而是整个支撑社会秩序、政治权威、伦理关系和文化意义的制度失去效力。诸子百家正是在这种具体问题刺激下，提出各自的解决方案。

### 【思想解释】儒、道、墨、法对礼乐危机的回应

儒家基本肯定礼乐制度。孔子主张恢复周文、恢复礼乐，并提出哲学根据：仁义，尤其是「仁」，是礼乐的内在根据。礼乐不是空洞形式，而是仁爱之心的表达；人有爱并需要表达爱，所以需要礼乐制度来承载和传达。

道家基本反对礼乐制度。老师解释，道家认为礼乐会成为束缚生命的外在规范，例如繁文缛节、客套、礼貌和无谓应酬会让生命不自由。道家追求生命自由，希望人能达到逍遥自在的境界，所以会批判礼乐制度，并提出另一套想法。

墨家也反对礼乐，但理由不同。墨家从实用主义、节用立场看礼乐，认为许多礼服、礼器、乐队、排场和仪式都耗费资源，没有实际价值，像是贵族或有闲阶级的娱乐，是浪费民力与财力。

法家也反对以礼乐治国。法家认为礼乐不足以治理乱世，因为人未必听从道德劝说；要治理国家应当用法、刑罚和制度，让人不得不服从。因此法家强调以法治国、以刑法治国。

这一段说明，儒、道、墨、法虽然立场不同，但都在回应「礼乐制度疲弊」这一共同问题。所以「周文疲弊」可以解释诸子百家为什么在春秋战国集中出现。

### 【历史/文化解释】南方楚地文化与道家气质

最后一部分专门针对道家。老师说，道家思想的产生还有一个特殊背景：它从南方楚地文化中孕育而成，受到南方楚地文化精神影响。

这里的「南方」是古代中国的南方，不是今天以香港、广东为中心理解的南方。课堂中特别提醒，老子、庄子与河南一带有关；按今天位置看似偏北，但在当时的文化地理中可放入「南方楚地」脉络。河南以北才更接近课堂所说的北方。

老师用地理环境解释南北文化风格：北方平原较多，文化较直率、朴实、质朴、实在，偏阳刚；南方山多、云气多，文化较虚无缥缈、浪漫、富想象、委婉柔软，偏阴柔。老师也说明，今天这种分别已经不明显，但古代较为突出。

文学例子是《诗经》和《楚辞》。《诗经》代表较多北方风格，如「关关雎鸠，在河之洲。窈窕淑女，君子好逑」表达爱情，语言较平实直接。《楚辞·山鬼》也写爱情，但「若有人兮山之阿，被薜荔兮带女萝，既含睇兮又宜笑，子慕予兮善窈窕」一类语言更柔婉、浪漫、神秘，充满山林、神灵和想象。

思想例子是《论语》和《道德经》。孔子《论语》开篇「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」道理平实、朴素、贴近日常实践。《道德经》第二十一章则说「道之为物，惟恍惟惚；惚兮恍兮，其中有象；恍兮惚兮，其中有物；窈兮冥兮，其中有精；其精甚真，其中有信」。老师借「恍惚」「窈冥」说明道家语言较玄妙、神秘，不像儒家表达那么平实。

因此，道家思想的出现既有春秋战国共同的社会历史背景，也有南方楚地文化的精神气质作为特殊背景。

## 关键概念

- 诸子出于王官：汉代以来解释诸子百家来源的传统说法，认为各家思想来自周代掌握不同专业知识的王官。
- 王官：贵族官僚体系中掌握、保存、传承专门知识的官员。
- 史官：掌管历史和典籍的官员；传统说法认为道家出于史官，因为史官能观察文明兴衰和社会运行规律。
- 士人/民间知识分子：冯友兰修正说中的关键群体，指王官知识流落民间后形成的读书人、教师或知识分子。
- 隐士：冯友兰说中道家的民间来源；不是普通隐居者，而是有学问、看透时代而退隐的人。
- 救世之弊：胡适对诸子来源的解释，认为诸子思想是为回应社会现实弊病而产生。
- 周文疲弊：牟宗三对「救世之弊」的具体化，指周代典章文化、礼乐制度发生流弊和衰败。
- 礼崩乐坏：春秋战国时期礼乐制度失效，社会秩序和价值系统发生危机。
- 南方楚地文化：老师用来解释道家风格的文化背景，特点是浪漫、玄妙、富想象、较阴柔。
- 恍惚/窈冥：课堂引用《道德经》第二十一章说明道的不可把握和道家语言的玄妙气质。
