# 《逍遥游》概念树状图

资料依据：`review/期末考试复习手册.md` 与 `lectures/organized_notes/lecture_20.md` 至 `lecture_23.md`。

## 总树

- 《逍遥游》
  - 总主题：精神自由
    - 逍遥：自由自在、无拘无束。
    - 游：在人、事、物之间自由活动，不被固定条件束缚。
    - 不是现代政治自由：不是言论自由、学术自由、宗教自由等制度性自由。
    - 核心：无待。
      - 不把生命安顿、幸福、快乐寄托在不可控制的外在条件上。
      - 指向得道境界与精神上的真正自由。
  - 第一层展开：鲲鹏寓言
    - 文本形象
      - 北冥有鱼，名鲲。
      - 鲲化为鸟，名鹏。
      - 鹏徙南冥，南冥为天池。
      - 水击三千里，抟扶摇而上九万里。
      - 去以六月息。
    - 相关概念
      - 北冥、南冥、天池：营造超出日常经验的宏大空间。
      - 《齐谐》：志怪之书，增强鲲鹏故事的神奇感。
      - 水击三千里：表现鹏起飞的壮阔。
      - 抟扶摇：乘旋风盘旋而上。
      - 六月息：可解释为六月大风，也可解释为飞六个月才停止；课堂保留两说。
      - 野马、尘埃：从高空俯视地面时所见的气息流动与尘埃景象。
      - 天之苍苍：通过地上看天、天上看地的转换，表现视角问题。
    - 思想意义
      - 鲲鹏象征宏大气魄、高远理想、广阔视野与求道境界。
      - 但鲲鹏仍不是最终逍遥。
        - 水不厚不能负大舟。
        - 风不厚不能负大翼。
        - 鹏必须依赖厚风，所以仍然有待。
  - 第二层展开：小大之辨
    - 小
      - 蜩、鸴鸠：小飞虫、小鸟，嘲笑鹏飞九万里。
      - 斥鴳：只在蓬蒿之间腾跃，也嘲笑鹏。
      - 朝菌：不知晦朔，生命尺度短。
      - 蟪蛄：不知春秋，生命尺度短。
      - 彭祖：在人间虽长寿，但相对冥灵、大椿仍属小。
      - 世俗成就
        - 知效一官：才智能胜任一官。
        - 行比一乡：行为能合于一乡。
        - 德合一君：德行能合于国君。
        - 而征一国：能力能取信一国。
        - 现代例子：拿 A、毕业、求职、做 CEO、发财、做高官等。
    - 大
      - 鲲、鹏：大体量、大气魄、大视野。
      - 冥灵、大椿：大生命尺度。
      - 求道境界：高远理想、深远境界、广阔视野。
    - 关键判断
      - 小大不只是体积、寿命、能力差别，而是境界高低。
      - “小知不及大知，小年不及大年”的“不及”表示小低于大。
      - 小者嘲笑大者，说明小境界不能理解大境界。
      - 但“大”也不等于最终无待；鲲鹏虽大，仍待于风。
  - 第三层展开：有待与无待
    - 有待
      - 意思：有所等待、有所依赖。
      - 凡快乐、幸福、成就、身份、价值依赖不可控外在条件，就是有待。
      - 例子
        - 当官有待于上司任用。
        - 做宰相有待于国君信任。
        - 做皇帝有待于民心。
        - 学生拿 A 有待于老师评分。
        - 求职有待于公司聘用。
        - 列子飞行有待于风。
      - 问题
        - 条件存在时才快乐，条件消失时快乐也消失。
        - 喜怒哀乐被环境和他人决定，所以不是真正自由。
    - 无待
      - 意思：不依赖不可控制的外在条件来安顿生命。
      - 真正表述
        - 乘天地之正。
          - 课堂解释为顺应道的本体。
        - 御六气之辩。
          - 六气：阴、阳、风、雨、晦、明。
          - 指顺应并驾驭万物变化。
        - 以游无穷。
          - 在无穷人、事、物之间自由活动。
        - 彼且恶乎待哉。
          - 还要依赖什么呢？表示无所待。
  - 第四层展开：高于世俗但仍未无待的人物
    - 宋荣子
      - 特点
        - 举世誉之而不加劝。
        - 举世非之而不加沮。
        - 定乎内外之分。
        - 辩乎荣辱之境。
      - 境界
        - 高于一般追逐世俗名利的人。
        - 不轻易被赞誉、批评、荣辱左右。
      - 限制
        - “犹有未树”：还没有真正达到无待。
        - 仍停在内/外、荣/辱的相对结构中。
        - 只要还强调不受外界影响，心中仍有外界作为对立面。
    - 列子
      - 特点
        - 御风而行，泠然善也，旬有五日而后反。
      - 境界
        - 比普通人高明，似乎自由。
      - 限制
        - “犹有所待”：仍依赖风。
        - 有风才能飞，无风则不能飞，所以仍不是无待。
  - 第五层展开：无待的三个方面
    - 总纲：至人无己，神人无功，圣人无名。
      - 课堂倾向：三者是无待的三个平等方面，不必严格分成高低等级。
      - 共同意思：不是没有己、没有功、没有名，而是不待于己、不待于功、不待于名。
    - 圣人无名
      - 核心
        - 圣人不把幸福、生命价值寄托于名声。
        - 也可引申为圣人无利：不把生命安顿寄托于大利。
      - 寓言：尧让天下于许由
        - 尧：古代圣王，愿把天下让给许由。
        - 许由：有道隐士，拒绝天下。
        - 名者，实之宾也。
          - 名声只是实功的附属物，不值得为了名而接受天下。
        - 鹪鹩巢于深林，不过一枝。
        - 偃鼠饮河，不过满腹。
          - 人真正所需有限，不必被天下之名利牵引。
        - 庖人虽不治庖，尸祝不越樽俎而代之。
          - 不越过自己的本分去替尧治理天下。
      - 考点判断
        - 名位越大，不等于生命越自由。
        - 圣人无名不是没有名声，而是不待于名声。
    - 神人无功
      - 核心
        - 神人不把幸福快乐依赖在某一特定事功上。
        - 不是没有成就，而是不执着特定事功。
        - 与老子“无为而无不为”相通。
      - 寓言一：肩吾、连叔、接舆与藐姑射神人
        - 肩吾听接舆之言，认为“大而无当，往而不返”，不近人情。
        - 连叔回应：盲者不能看五彩，聋者不能听钟鼓；智慧上也有聋盲。
        - 藐姑射神人
          - 肌肤若冰雪，淖约若处子。
          - 不食五谷，吸风饮露。
          - 乘云气，御飞龙，游乎四海之外。
          - 其神凝，使物不疵疠而年谷熟。
        - 神人与万物为一
          - 旁礡万物以为一。
          - 不是弊弊焉以天下为事。
          - 不忙于某一项事务，却能使万物各得其所。
      - 寓言二：宋人资章甫而适诸越
        - 宋人带礼冠到越国贩卖。
        - 越人断发文身，无所用之。
        - 含义
          - 若把幸福寄托在一项特定事业或生意上，条件不合就会痛苦。
          - 不是说不要做事，而是不要执着于某一固定事功。
      - 寓言三：尧见四子
        - 尧治理天下后见藐姑射四子。
        - 窅然丧其天下。
        - 可理解为被神人境界震动，发现圣王政治仍不及神人境界；也可理解为融入神人境界而忘掉天下。
      - 考点判断
        - 神人无功不是没有功。
        - 是不待于功、不执着特定事功。
    - 至人无己
      - 核心
        - 无己就是无我。
        - 不是取消自我、消灭人格、没有个性。
        - 是有我而不执着固定自我结构。
      - 课堂比喻：水
        - 水遇圆成圆，遇方成方，遇三角成三角。
        - 水仍是水，对象仍是对象，事情完成而双方不受伤。
        - 至人能随物自然相应，不以固定自我强压对象。
      - 寓言一：惠子大瓠
        - 惠子得到五石之瓠。
        - 盛水太重，剖开做瓢又太大无处可容，于是认为无用而打碎。
        - 庄子批评：夫子固拙于用大矣。
      - 关联故事：不龟手之药
        - 同一种药能防手冻裂。
        - 宋人用于洴澼絖，只得小利。
        - 客人用于吴越水战，因功裂地受封。
        - 关键句：能不龟手，一也；或以封，或不免于洴澼絖，则所用之异也。
        - 含义：同一事物因使用方向不同，可以有小用或大用。
      - 大瓠的大用
        - 可虑以为大樽，浮乎江湖。
        - 大樽可理解为腰舟、渡水浮具。
        - 大瓠不能作普通容器，不等于无用。
      - 蓬之心
        - 心思闭塞、不通达、不开阔。
        - 惠子的问题不是大瓠无用，而是心被普通用途限制。
      - 寓言二：大树
        - 惠子说大树为樗，树干臃肿不中绳墨，小枝卷曲不中规矩。
        - 匠人不顾，所以看似无用。
        - 庄子回应狸狌与犛牛。
          - 狸狌能捕鼠，却容易中机关、死于网罟。
          - 犛牛巨大，不能捕鼠，却有自己的存在方式。
        - 无何有之乡、广莫之野
          - 把大树种在那里，人可彷徨乎无为其侧，逍遥乎寝卧其下。
        - 无用之用
          - 大树正因无所可用，所以不被砍伐。
          - 看似无用，反而能保全生命、安顿身心。
      - 考点判断
        - 至人无己不是没有人格。
        - 是不执着小/大、有用/无用、有我/无我的相对限制。
        - 无用之用使生命从单一工具价值中解放出来。

## 速记版

- 《逍遥游》主题：精神自由。
- 精神自由核心：无待。
- 鲲鹏：大境界开端，但仍待于风。
- 小大之辨：小是世俗境界，大是求道境界；小低于大，但大仍未必无待。
- 有待：幸福、成就、身份依赖外在条件。
- 无待：不把生命安顿交给不可控条件。
- 宋荣子：不受毁誉左右，但仍在内外荣辱相对中。
- 列子：能御风，但仍待于风。
- 圣人无名：不待于名利；许由拒绝天下。
- 神人无功：不待于特定事功；藐姑射神人、宋人卖章甫。
- 至人无己：不执着固定自我；大瓠、大树、无用之用。
